Requisitos Funcionales

Usuarios Arrendadores

Creación de Cuenta (HUA1)

POST /arrendadores/registro para registrar un nuevo usuario arrendador en el sistema.

Activación de Cuenta (HUA2)

GET /arrendadores/activacion/{token} para activar la cuenta del arrendador a través del token recibido por correo electrónico.

Autenticación de Usuario (HUA3)

POST /auth/login para que los arrendadores inicien sesión en el sistema.

Ver Mis Propiedades (HUA4)

GET /propiedades/mis-propiedades para que los arrendadores vean un listado de sus propiedades.

Crear Propiedad (HUA5)

POST /propiedades/nueva para crear una nueva propiedad en el sistema.

Editar Propiedad (HUA6)

PUT /propiedades/editar/{idPropiedad} para editar los detalles de una propiedad existente.

Subir Foto de la Propiedad (HUA7)

POST /propiedades/{idPropiedad}/subir-foto para subir fotos a una propiedad específica.

Desactivar Propiedad (HUA8)

PATCH /propiedades/desactivar/{idPropiedad} para desactivar una propiedad.

Ver Solicitudes de Arriendo (HUA9)

GET /solicitudes/{idPropiedad} para ver las solicitudes de arriendo de una propiedad.

Aceptar Solicitud de Arriendo (HUA10)

PATCH /solicitudes/aceptar/{idSolicitud} para aceptar una solicitud de arriendo.

Cancelar Solicitud de Arriendo (HUA11)

PATCH /solicitudes/cancelar/{idSolicitud} para cancelar una solicitud de arriendo.

Calificar Arrendatario (HUA12)

POST /calificaciones/{idArrendatario} para calificar a un arrendatario después de una estancia.

Usuarios Arrendatarios

Creación de Cuenta (HUA13)

POST /arrendatarios/registro para registrar un nuevo usuario arrendatario.

Activación de Cuenta (HUA14)

GET /arrendatarios/activacion/{token} similar al proceso de los arrendadores.

Autenticación de Usuario (HUA15)

POST /auth/login igualmente para que los arrendatarios inicien sesión.

Buscar Propiedades (HUA16)

GET /propiedades/buscar para buscar propiedades disponibles para arrendar.

Solicitar Arriendo (HUA17)

POST /solicitudes/nueva para que un arrendatario solicite el arriendo de una propiedad.

Ver Mis Solicitudes de Arriendo (HUA18)

GET /solicitudes/mis-solicitudes para que los arrendatarios revisen el estado de sus solicitudes.

Pagar Arriendo (HUA19)

POST /pagos/realizar para realizar el pago de un arriendo aceptado.

Calificar Finca (HUA20)

POST /calificaciones/{idFinca} para que los arrendatarios califiquen una finca después de su estancia.
