Casos de Prueba para el Frontend

Registro de Usuario

Objetivo: Verificar que el formulario de registro captura y envía correctamente la información al backend.

Pasos:

  • Acceder a la página de registro.
  • Completar todos los campos requeridos con datos válidos.
  • Enviar el formulario.

Resultado esperado: El usuario recibe una confirmación de registro y un email de activación.

Inicio de Sesión

Objetivo: Comprobar que el sistema permite el inicio de sesión con credenciales válidas.

Pasos:

  • Acceder a la página de inicio de sesión.
  • Introducir credenciales válidas de un usuario previamente registrado.
  • Enviar el formulario.

Resultado esperado: El usuario es redirigido al dashboard principal.

Visualización de Propiedades

Objetivo: Confirmar que los usuarios pueden ver un listado de propiedades disponibles.

Pasos:

  • Iniciar sesión como arrendatario.
  • Navegar a la sección de propiedades disponibles.

Resultado esperado: Se muestra un listado de propiedades disponibles para arrendar.

Envío de Solicitud de Arriendo

Objetivo: Verificar el proceso de envío de una solicitud de arriendo para una propiedad.

Pasos:

  • Como arrendatario, seleccionar una propiedad del listado.
  • Rellenar el formulario de solicitud de arriendo con los datos requeridos.
  • Enviar la solicitud.

Resultado esperado: Se muestra un mensaje de confirmación de que la solicitud ha sido enviada.
